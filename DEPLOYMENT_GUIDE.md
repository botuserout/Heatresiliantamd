# 🌐 How to Deploy Your Dashboard Online (Free)

Follow these 3 simple steps to put your Ahmedabad HeatResilient project on the web!

## 1. Upload to GitHub
- Create a new (Free) account on **[GitHub.com](https://github.com)** if you don't have one.
- Create a new Repository named `Ahmedabad-Heatwave-Analysis`.
- Upload all these files from your folder to that repository:
    - `app.py`
    - `requirements.txt`
    - `data/` (the whole folder with the CSV)
    - `ahmedabad_colab_notebook.py`

## 2. Connect to Streamlit Cloud
- Go to **[share.streamlit.io](https://share.streamlit.io)**.
- Log in using your GitHub account.
- Click **"Create app"**.

## 3. Launch!
- Select your **Repository** (`Ahmedabad-Heatwave-Analysis`).
- Select the **Main file path**: `app.py`.
- Click **"Deploy!"**.

---

### 🎉 What happens next?
- Streamlit will spend 1-2 minutes "cooking" your app (installing the libraries from `requirements.txt`).
- You will get a unique URL (like `ahmedabad-heat.streamlit.app`).
- **You can share this link with your teacher!** She can open it on her browser or phone without installing anything.

> **Tip:** If Ma'am asks how you deployed it, tell her: *"I used Streamlit Community Cloud and linked it to my GitHub repository for seamless deployment."*
